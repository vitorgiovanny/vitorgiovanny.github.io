# Conteúdo por arquivo Markdown

Cada menu abaixo lê automaticamente os arquivos .md da sua coleção. Para publicar um item novo, basta criar um arquivo na pasta correta com front matter Jekyll.

</> Desafios {{ site.desafios.size }} arquivo(s) Q Computação Quântica {{ site.computacao_quantica.size }} arquivo(s) A Arquiteturas {{ site.arquiteturas.size }} arquivo(s) DP Design Patterns {{ site.design_patterns.size }} arquivo(s)

Como adicionar conteúdo

Crie um arquivo Markdown na pasta da coleção desejada. O card aparece automaticamente na página da seção após o build do Jekyll.

Pastas:
  • _desafios
  • _computacao_quantica
  • _arquiteturas
  • _design_patterns
